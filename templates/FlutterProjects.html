Astrodrishti (july 2020 - October 2020)

It is a cross platform mobile app which provides user his birth kundli using api and also using various conditions provided by astrologer it will predict the good and bad in kundli . Also person can login using google and save kundli in firebase.

FitZen (March 2021 - Ongoing)

It is a cross platform Fitness mobile app which provides features like workouts, transformation plans, Diet plans, Fitness Calculator and LogBook. User can also register and save his/her data. Logbook will provide track of users progress and can represent it graphically.
